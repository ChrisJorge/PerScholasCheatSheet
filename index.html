Key Terminology

Project

An Individual or collaborative enterpries that is carefully planned to achieve a particular outcome

  • Specific
  • Requires one-off performance
  • Requires planning
  • Large enough to need control mechanisms in place
  • Is not a business as usual activity

Brownfield Project

  • System conversion

Greenfield project

  • New implementation

Gantt Chart

Visual tool that shows what must be done and when it should be done by

Project Management

Required to ensure effective management of all activites, ensuring sucessful project completion.

  • On Time
  • Within Budget
  • Appropriate Quality

Project Manager

Responsible for managing the process and the activities of project management.

  • Planning all activities
  • Overseeing all execution
  • Monitoring progress
  • Reporting to stakeholders

Project Charter

Document that provides formal approval to proceed as a project

  • Scope
  • Objective
  • Measurments and assumptions
  • Restrictions
  • Key risks
  • Major participants
  • Project authorities
  • Expected or approved costs
  • Timeframes - Deliveries and milestones
  • Expectations of the quality of deliverables

Project Schedule

Used for documenting and detailing key milestones (defined points showing project progress)

  • Can be displayed as a Gantt Chart or list
  • When time arrives, progress will be measured and compared against milestone
